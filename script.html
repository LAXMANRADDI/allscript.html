HTML → structure CSS → design JS → logic (click, validate, change page content) JS in HTML: Inline Internal External 1. Variables variables to store values. let name = "Laxman"; const pi = 3.14; var age = 20; Use let and const only. Avoid var. --- 2. Data Types String → "hello" Number → 10, 3.14 Boolean → true/false Array → [] Object → {} Null → nothing Undefined → not yet assigned __ 3.Conditions let marks = 85; if(marks >= 90){ console.log("A+"); } else if(marks >= 70){ console.log("B"); } else { console.log("C"); } ____ 4. Functions function add(a, b) { return a + b; } console.log(add(5, 10)); Arrow function const add = (a, b) => a + b; --- 5. Arrays let arr = [10, 20, 30]; console.log(arr[0]); arr.push(40); arr.pop(); --- 6. Objects let student = { name: "Laxman", age: 20, marks: 95 }; console.log(student.name); DOM = Document Object Model It represents your HTML page as a tree of objects that JavaScript can control. Selecting Elements (Most Important Part) 1. getElementById document.getElementById("title") 2. getElementsByClassName document.getElementsByClassName("box") 3. getElementsByTagName document.getElementsByTagName("p") 4. querySelector → selects FIRST match document.querySelector(".container") 5. querySelectorAll → selects ALL matches document.querySelectorAll("p") --- 3. Changing Content innerText document.getElementById("title").innerText = "Welcome!"; innerHTML (renders HTML) document.getElementById("title").innerHTML = "Bold Text"; textContent Shows hidden text also. element.textContent = "New text"; --- 4. Changing Styles (CSS using JS) document.getElementById("title").style.color = "red"; document.getElementById("title").style.fontSize = "40px"; --- 5. Adding & Removing Classes element.classList.add("active"); element.classList.remove("hide"); element.classList.toggle("dark-mode"); --- 6. Creating New Elements (Important) Step 1 → create element let div = document.createElement("div"); Step 2 → set content div.innerText = "I am new!"; Step 3 → attach it into HTML document.body.appendChild(div); For inserting inside another element: parent.appendChild(child); parent.prepend(child); parent.insertBefore(child, reference); --- 7. Removing Elements let box = document.getElementById("box"); box.remove(); --- 9. Adding Events (3 Methods) Method 1 → Inline Method 2 → Property Event button.onclick = function(){ console.log("Button clicked"); } Method 3 → addEventListener (BEST) document.getElementById("btn").addEventListener("click", function(){ alert("Clicked!"); }); --- 10. SUPER IMPORTANT: Event Object document.addEventListener("click", function(event){ console.log(event.target); }); event.target tells which element fired the event. --- 11.
